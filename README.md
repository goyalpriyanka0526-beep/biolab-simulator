# 🧬 BioLab Simulator

A Virtual Molecular Biology Laboratory for Biotechnology Students.

## Features

- PCR Simulator
- Gel Electrophoresis Simulator
- DNA Extraction Simulator
- Interactive Web Interface
- Educational Feedback and Explanations

## Technologies Used

- Python
- Streamlit
- Replit
